Introduction

Some CSS properties can be combined into shorthand properties, to save writing multiple properties.

/* longhand properties */
li {
  list-style-type: disc;
  list-style-image: none;
  list-style-position: inside;
}


/* shorthand property */
li { list-style: disc none inside; }

2. Order of values

Some shorthand properties can be used to define all four edges of an element.

These include margin, padding, border-color, border-width, border-style.

These shorthand properties can be defined using either one, two, three or four values.

p { margin: 1em; }
p { margin: 1em 2em; }
p { margin: 1em 2em 3em; }
p { margin: 1em 2em 3em 4em; }

The order that values are specified for two, three and four values is critical!

If only one value is defined, this value defines all four edges of the element: (top/right/bottom/left).

p { margin: 1em; }

If two values are defined, the first value defines the top and bottom edges of the element, and the second value defines the left and right edges of the element.

p { margin: 1em 2em; }

If three values are defined, the first value defines the top edge, the second value defines the left and right edges, and the third value defines the bottom edge.

p { margin: 1em 2em 3em; }

If four values are defined, they define in order, top, right, bottom and left edge.

p { margin: 1em 2em 3em 4em; }

So, the four values are always defined clockwise - starting at the top.

3. Initial values

All CSS properties have an initial value - a value that is used if the value is not specified by the author.

For shorthand properties, the initial values are normally based on the initial values of the longhand properties.

For example, the shorthand border property uses the longhand border-width, border-style and border-color properties.

The initial values for the border-width, border-style and border-color properties are:

/* initial values */
p { border-width: medium; }

p { border-style: none; }

p { border-color:
  [inherits the 'color' value];
}

Therefore the initial values for the border property are:

p { border:
  medium
  none
  [inherits the 'color' value];
}

4. Margin

The margin shorthand property is defined using one, two, three or four <margin-width> properties:

/* initial shorthand value */
p { margin: 
  <margin-width>{1,4};
}

The initial value for <margin-width> is 0. Therefore, the initial value for the shorthand margin property is also 0:

/* initial shorthand values */
p { margin: 0; }

The different shorthand value options are:

/* longhand */
p {
  margin-top: 1em;
  margin-right: 1em;
  margin-bottom: 1em;
  margin-left: 1em;
}

/* one value */
p { margin: 1em; }

/* longhand */
p {
  margin-top: 1em;
  margin-right: 2em;
  margin-bottom: 1em;
  margin-left: 2em;
}

/* two values */
p { margin: 1em 2em; }

/* longhand */
p {
  margin-top: 1em;
  margin-right: 2em;
  margin-bottom: 3em;
  margin-left: 2em;
}

/* three values */
p { margin: 1em 2em 3em; }

/* longhand */
p {
  margin-top: 1em;
  margin-right: 2em;
  margin-bottom: 3em;
  margin-left: 4em;
}

/* four values */
p { margin: 1em 2em 3em 4em; }
